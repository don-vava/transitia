TransitIA — IA de compagnonnage pour la reconversion professionnelle Ne pas réapprendre un métier. Traduire le sien. 
TransitIA est une solution ESS propulsée par l'IA générative qui accompagne les travailleurs en reconversion en temps réel, 
sans formation lourde. Plutôt que d'imposer un nouveau jargon, l'outil traduit les tâches et le vocabulaire du secteur d'arrivée dans les codes 
du métier d'origine — via une oreillette et une interface mobile.Exemple : un mécanicien qui rejoint 
une équipe RSE entend "contrôle technique bilan carbone" au lieu de "diagnostic GES". 
Il est opérationnel en 3 semaines, pas en 3 mois.
